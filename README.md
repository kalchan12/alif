# Alif PLC - Corporate Website

A premium, modern corporate landing page for **Alif PLC**, showcasing the company's dual expertise in high-quality manufacturing and cutting-edge technology solutions.

## 🚀 Key Features

-   **Dual-Division Focus**: Clear representation of the Manufacturing and Technology business arms.
-   **Modern Tech Stack**: Built with Next.js 15, Tailwind CSS v4, and Framer Motion for premium animations.
-   **Dark & Light Mode**: Seamless theme switching with persistence using `next-themes`.
-   **Responsive Design**: Fully optimized for mobile, tablet, and desktop viewports.
-   **Performance-First**: Optimized production build with zero errors.

## 🛠 Tech Stack

-   **Framework**: [Next.js](https://nextjs.org/) (App Router)
-   **Styling**: [Tailwind CSS v4](https://tailwindcss.com/)
-   **Animations**: [Framer Motion](https://www.framer.com/motion/)
-   **Icons**: [Lucide React](https://lucide.dev/)
-   **Theming**: [next-themes](https://github.com/pacocoursey/next-themes)

## 📂 Project Structure

```text
/src
  /app           - Next.js folder structure (layout, page, globals.css)
  /components
    /sections    - Modular landing page sections (Hero, Navbar, etc.)
    /ui          - Reusable UI elements (Button, Section, ThemeToggle)
  /lib           - Utility functions (cn merger)
```

## 🪜 Getting Started

### 1. Installation
Clone the repository and install dependencies:
```bash
npm install
```

### 2. Development
Start the development server:
```bash
npm run dev
```

### 3. Build
Create an optimized production build:
```bash
npm run build
```

## 🎨 Branding & Aesthetics

The design uses a **Slate & Blue** corporate palette:
-   **Light Mode**: Soft white background (`slate-50`) with deep navy text.
-   **Dark Mode**: Elegant deep dark background (`#0B0F19`) with premium card overlays.
-   **Typography**: Clean, professional sans-serif (Inter/Geist) for a tech-forward corporate feel.
